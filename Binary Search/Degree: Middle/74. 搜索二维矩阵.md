#### [74. 搜索二维矩阵](https://leetcode-cn.com/problems/search-a-2d-matrix/)

> - `row = mid / n`, `col = mid % n`

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        int m = matrix.length;
        if(m == 0){
            return false;
        }
        int n = matrix[0].length;
        int left = 0;
        int right = m * n - 1;
        while(left <= right){
            int mid = (left + right) >>> 1;
            int value = matrix[mid / n][mid % n];
            if(value == target){
                return true;
            }else if(value < target){
                left = mid + 1;
            }else if(value > target){
                right = mid - 1;
            }
        }
        return false;
    }
}
```

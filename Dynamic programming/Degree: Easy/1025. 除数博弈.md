#### [1025. 除数博弈](https://leetcode-cn.com/problems/divisor-game/)

```java
      public boolean divisorGame(int N) {
         //base case
        if(N <= 1) return false;
        if(N == 2) return true;
        boolean[]dp = new boolean[N+1];
          //初始化为false
        Arrays.fill(dp,false);
        dp[1] = false;
        dp[2] = true;

        for(int i = 3; i <= N; i++){
            for(int j = 1; j < i; j++){
                if(i % j == 0 && !dp[i-j]){
                    dp[i] = true;
                    break;
                }
            }
        }
        return  dp[N];
    }
```

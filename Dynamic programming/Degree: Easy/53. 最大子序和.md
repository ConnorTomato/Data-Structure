#### [53. 最大子序和](https://leetcode-cn.com/problems/maximum-subarray/)

> dp

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int len = nums.length;
        if(len == 0){
            return 0;
        }
        int[] dp = new int[len + 1];
        // base case
        dp[0] = nums[0];
        for(int i = 1; i < len; i++){
            //状态转移
            dp[i] = Math.max(nums[i],nums[i] + dp[i - 1]);
        }
        int res = Integer.MIN_VALUE;
        for(int i = 0; i < len; i++){
            res = Math.max(res,dp[i]);
        }
        return res;
    }
}
```

> 状态压缩

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int len = nums.length;
        if(len == 0){
            return 0;
        }
        // base case
        int dp_0 = nums[0];
        int dp_1 = 0;
        int res = dp_0;
        for(int i = 1; i < len; i++){
            //状态转移
          dp_1 = Math.max(nums[i],nums[i] + dp_0);
          dp_0 = dp_1;
          res = Math.max(res,dp_1);
        }

        return res;
    }
}
```

> 在线处理

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int maxSum = nums[0];
        int thisSum = 0;
        for(int i = 0; i<nums.length; i++){
            thisSum += nums[i];
            if(thisSum > maxSum)
                maxSum = thisSum;
            if(thisSum < 0)
                thisSum = 0;
        }
        return maxSum;
    }
}
```

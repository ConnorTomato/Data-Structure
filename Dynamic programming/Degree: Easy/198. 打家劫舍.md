#### [198. 打家劫舍](https://leetcode-cn.com/problems/house-robber/)

> 递推关系式`dp[i] = Math.max(dp[i - 1], nums[i - 1] + dp[i-2])`

```java
class Solution {
    public int rob(int[] nums) {
        int len = nums.length;
        if(len == 0){
            return 0;
        }

        int[] dp = new int[len + 1];
        dp[0] = 0;
        dp[1] = nums[0];
        for(int i = 2; i <= len; i++){
            dp[i] = Math.max(dp[i-1], nums[i-1] + dp[i - 2]);
        }
        return dp[len];
    }
}
```

> 空间优化

``` java
class Solution {
    public int rob(int[] nums) {
        int len = nums.length;
        if(len == 0){
            return 0;
        }

      int pre = 0;
      int cur = 0;
        for(int i : nums){
            int temp = Math.max(cur,pre + i);
            pre = cur;
            cur = temp;
        }
        return cur;
    }
}
```

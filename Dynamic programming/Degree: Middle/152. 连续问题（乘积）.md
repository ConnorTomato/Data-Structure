#### [152. 乘积最大子数组](https://leetcode-cn.com/problems/maximum-product-subarray/)

> 暴力解法

```java
class Solution {
    public int maxProduct(int[] nums) {
        if(nums.length < 2){
            return nums[0];
        }
        int Sum = nums[0];
     for(int i = 0; i < nums.length; i++){
         int tempSum = nums[i];
         Sum = Math.max(Sum,tempSum);
         for(int j = i; j < nums.length - 1; j++){
             tempSum *=  nums[j + 1];
             Sum = Math.max(Sum,tempSum);          
         }
     }
     return Sum;
    }
}
```

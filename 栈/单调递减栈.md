#### [739. 每日温度](https://leetcode-cn.com/problems/daily-temperatures/)

>暴力解法

```java
class Solution {
    public int[] dailyTemperatures(int[] T) {
        int len = T.length;
        if(len == 0){
            return new int[]{0};
        }
       int[] res = new int[len];
        for(int i = 0; i < len; i++){
             int cnt = 0;
            int pre = T[i];
            for(int j = i+1; j < len; j++){
                cnt++;
                if(pre < T[j]){
                    break;
                }
                if(j == len - 1){
                    cnt = 0;
                }
            }
            res[i] = cnt;
        }
        return res;
    }
}
```

> 单调递增栈

```java
 public int[] dailyTemperatures(int[] T) {
       Stack<Integer> stack = new Stack<>();
        int length = T.length;
        int[] result = new int[length];

        for (int i = 0; i < length; i++) {
            while (!stack.isEmpty() && T[i] > T[stack.peek()]) {
                int pre = stack.pop();
                result[pre] = i - pre;
            }
            stack.add(i);

        }
        return result;
    }
```
